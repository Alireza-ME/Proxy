
# :link:Send Proxy By BPT Proto 

Source Robot Send proxy to the channel based on the library BPT

Install source:

Step 1: Edit the config.php file

Step 2: Set on the Atuo-send.php file 1-minute cron Job

Step 3 : Set Web Hook On the index.php File

Last step: Admin your robot inside the channel and enjoy (:

Commands -> 1-/Send 2-/MultiSend

/Send  : Send a one proxy to the channel

/MultiSend : Send 10 proxies to the channel
